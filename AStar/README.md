# A Star Algorithm Simulator 

User Interface 



Map Generation Stage 

- Make Wall 
- Remove Wall 
- Press Enter to move next stage 


Start and End Point Selection Stage 

- Select Start Point
- Select End Point
- Press Enter to move next stage 

Run A Star Algorithm Stage 

- Press Space to go one step forward
- Press Backspace to go one step backward
- Press Enter to auto run to the end
